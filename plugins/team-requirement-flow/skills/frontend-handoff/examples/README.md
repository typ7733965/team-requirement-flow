# 前端接入文档示例

本目录是 `frontend-handoff` Skill 的**虚构示例产出**——配套「短链接服务」的前端接入步骤。

> 示例与任何真实项目无关，仅用于演示 Skill 应输出的文档结构与文风。

## 文件

- `前端接入短链接服务步骤.md` —— 完整 10 节前端接入文档

## 阅读要点

1. **第一节 全景流程图** —— 阶段 A/B/C 一目了然
2. **第五节 接口速查表** —— URL / 方法 / 鉴权 / 调用时机一张表
3. **第六节 错误码与前端处理对照表（★ 必读）** —— 字面量 + 接口 + 前端建议
4. **第七节 硬编码约定（★ 必读）** —— 单独成节，避免被前端跳过（这是联调最常见翻车点）
5. **第八节 联调 FAQ** —— 现象 → 原因 → 排查
6. **第九节 接入 Checklist** —— 阶段 A/B/C 自检清单

## 注意

虚构示例中默认 success code 为 `0`、错误码字面量为 `SL_*`，仅作演示。

真实项目的 Skill 输出**必须 Grep 源码**确认这些值，**不要抄设计方案**——这是 frontend-handoff 的核心质量要求：「只信源码不信文档」。

## 配套阅读

参考同需求的 [设计方案示例](../../design-doc/examples/)，理解从「设计 → 前端文档」的完整链路。
