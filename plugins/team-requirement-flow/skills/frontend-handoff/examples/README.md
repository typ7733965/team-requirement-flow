# 前端接入文档示例

本目录是 `frontend-handoff` Skill 的「真实产出示例」，来自 game-operation 项目「用户端登录认证组件」需求。

## 文件

- `前端接入登录组件步骤.md` —— 完整 11 节前端接入文档

## 阅读要点

参考以下做法：

1. **第一节 全景流程图** —— ASCII 时序图，标注 用户/前端/网关/后端 四列
2. **第五节 接口速查表** —— 一张表把所有接口的 URL/方法/鉴权/调用时机列清楚
3. **第六节 错误码与前端处理对照表（★ 必读）** —— 字符串字面量 + 触发接口 + **前端建议处理方式**
4. **第七节 设备号 / 国家区号** —— 「硬编码约定」单独成节并标 ★ 必读，避免被前端跳过
5. **第八节 常见联调问题与排查** —— 现象 → 可能原因 → 排查步骤
6. **第九节 接入 Checklist** —— 阶段 A / B / C 自检清单，前端逐条勾掉就能发版

## 注意

文档中所有错误码字符串与 success code (`20000`) 都是 Grep 源码确认过的，**不是凭设计方案抄的**。

这是 frontend-handoff Skill 的核心质量要求：**只信源码不信文档**。
